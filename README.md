# SafeGuard - Women Safety Alert System

🛡️ **SafeGuard** is a comprehensive women safety application designed to provide immediate emergency response and location tracking capabilities. The system combines hardware device integration with a web-based dashboard for real-time monitoring and alert management.

## 🚨 Features

### Core Safety Features
- **Emergency SOS Button**: One-touch emergency alert system
- **Real-Time GPS Tracking**: Automatic location detection and sharing
- **Emergency Contact Notifications**: Instant SMS/Email alerts to predefined contacts
- **Live Alert Dashboard**: Real-time monitoring of active alerts
- **Location Mapping**: Visual representation of incident locations

### User Management
- **Secure Authentication**: User login and registration system
- **Device Integration**: Hardware device connectivity status
- **Emergency Contact Management**: Add and manage emergency contacts

### Alert Management
- **Alert History**: Track all past emergency incidents
- **Status Monitoring**: Active, resolved, and pending alert states
- **Automatic Resolution**: Configurable auto-resolution timers

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with responsive design
- **Geolocation**: Browser Geolocation API
- **Real-time Updates**: JavaScript event-driven architecture

## 🚀 Getting Started

### Prerequisites
- Modern web browser with JavaScript enabled
- Internet connection for geolocation services
- Hardware safety device (optional, for full functionality)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/women-safety.git
   cd women-safety
   ```

2. **Open in Browser**
   - Open `login.html` in your web browser
   - No additional setup required - it's a client-side application

### Usage

1. **Register/Login**: Create an account or login to access features
2. **Add Emergency Contacts**: Configure who receives alerts
3. **SOS Activation**: Press the emergency button when in danger
4. **Monitor Alerts**: View active alerts on the dashboard

## 🔧 Configuration

### Emergency Contacts
- Add up to multiple emergency contacts
- Supports SMS and email notifications
- Contacts are stored locally in browser

### Alert Settings
- Auto-resolution timer (default: 2 minutes)
- Location accuracy settings
- Notification preferences

## 📱 How It Works

1. **Detection**: Hardware device or manual trigger detects emergency
2. **Location**: GPS coordinates are captured automatically
3. **Notification**: Emergency contacts receive instant alerts
4. **Tracking**: Location is displayed on the live dashboard
5. **Response**: Authorities can track and respond to the incident
